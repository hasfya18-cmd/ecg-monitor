Integrasi Live (Ngrok)

Untuk menghubungkan smartwatch sungguhan ke web app ini:

  1. Jalankan ngrok http 5000 di terminal.
  2. Arahkan request POST dari smartwatch/aplikasi ke [URL_NGROK]/api/ecg_device.
  3. Kirim payload JSON: {"values": [0.5, 0.6, ...]} berisi minimal 187 sample.

ℹ️ Cara Kerja Sistem

1

⌚ Akuisisi Sinyal (Perangkat IoT)

Smartwatch (atau Wearable EKG) merekam sinyal jantung pasien secara kontinu dari lokasinya masing-masing.

2

🌐 Transmisi Internet/Cloud

Sinyal dikirim via jaringan nirkabel jarak jauh (Wi-Fi/Seluler) ke Cloud Server. Dalam simulasi ini, kita menguji ketahanan AI terhadap koneksi buruk (Noise dan Packet Loss).

3

🧠 Klasifikasi AI

Setiap detak jantung (beat) dianalisis menggunakan model AI — CNN (Deep Learning) atau SVM (Machine Learning) — untuk mendeteksi aritmia.

4

🖥️ Monitoring Real-Time

Hasil klasifikasi ditampilkan secara real-time di dashboard ini. Jika terdeteksi abnormal, sistem memberikan alert.

⚠️ DISCLAIMER: Sistem ini merupakan simulasi untuk keperluan penelitian / skripsi dan BUKAN alat diagnosa medis resmi. Hasil klasifikasi AI tidak boleh digunakan sebagai dasar keputusan klinis tanpa pengawasan tenaga medis profesional.
